ANNOTTA — Íconos

Generador de íconos para la PWA. Solo abrí este archivo en el navegador.

Instrucciones:
1. Hacé clic en los dos botones de arriba para descargar los íconos.
2. Creá una carpeta icons/ dentro de tu proyecto.
3. Mové los dos archivos descargados a esa carpeta.
4. ¡Listo! Este archivo generate-icons.html no se sube a Vercel — es solo una herramienta local.
